# mmd_to_funscript
Blender Addon

blender_mmd_tools
https://github.com/MMD-Blender/blender_mmd_tools

블렌더에 추가해서 MMD의 허리 기울기를 funscript로 추출합니다.


# 글로벌 옵션

### 저장 경로

생성된 `.funscript` 파일이 저장될 폴더를 지정합니다.

### 파일 이름

출력 파일의 기본 이름을 지정합니다.

예시:

* motion.pitch.funscript
* motion.roll.funscript
* motion.twist.funscript

---

### 프레임 간격

몇 프레임마다 데이터를 추출할지 설정합니다.

* 낮을수록 정확도 ↑
* 높을수록 파일 크기 ↓, 처리 속도 ↑

예시:

* 1 = 모든 프레임 추출
* 5 = 5프레임마다 추출

---

### 추출 방식 (Tracking Mode)

#### Local

캐릭터 자체를 기준으로 회전을 측정합니다.

* 몸이 회전해도 상대적인 움직임 유지
* MMD 모션에 일반적으로 권장

#### Global

월드 좌표계(화면 기준)를 사용합니다.

* 캐릭터가 어느 방향을 보든 절대 방향 기준으로 계산

---

### 경계 지연 추적 사용

설정 범위를 초과하는 움직임이 발생할 때 즉시 끝값으로 고정하지 않고 부드럽게 따라갑니다.

효과:

* 갑작스러운 튐 감소
* 기계적인 움직임 완화

---

### 지연 추적 프레임

지연 추적 속도를 설정합니다.

* 낮음 = 빠르게 따라감
* 높음 = 천천히 따라감

---

# 범위 및 스캔 옵션

### 자동 범위 최적화 (Auto-Range)

애니메이션 전체를 분석하여 실제 사용된 회전 범위를 자동 계산합니다.

예시:

실제 움직임:
-30° ~ +30°

자동 계산:
-30° → 0
+30° → 100

따라서 작은 움직임도 전체 스트로크를 활용하게 됩니다.

---

### 전체 프레임 정밀 스캔

프레임 간격을 무시하고 모든 프레임을 검사합니다.

특히:

* 뒤돌기 보정
* 90도 반전
* 자동 범위 계산

의 정확도를 높여줍니다.

권장 상황:

* 복잡한 회전 모션
* 180도 이상 회전 모션

---

### 범위 지정 방식

#### 전체 범위 크기 (Width)

중심을 기준으로 대칭 범위를 생성합니다.

예시:

90°

↓

-45° ~ +45°

---

#### 최소/최대 직접 지정

사용자가 직접 범위를 지정합니다.

예시:

-20° ~ +70°

---

# 축별 옵션

Pitch / Roll / Twist 각각 동일한 방식으로 동작합니다.

---

## 활성화

해당 축의 Funscript 생성 여부를 결정합니다.

예시:

* Pitch만 ON → Pitch 파일만 생성
* Pitch + Roll ON → 2개 파일 생성

---

## 방향 전환

출력값을 반전합니다.

기본:

왼쪽 = 100
오른쪽 = 0

반전:

왼쪽 = 0
오른쪽 = 100

---

## 90도 반전 (Ping-Pong)

90도를 넘어가도 값이 튀지 않도록 접어서 계산합니다.

예시:

회전 각도

0°
45°
90°
135°
180°

출력

0
50
100
50
0

즉 왕복(Ping-Pong) 형태로 동작합니다.

권장:

* 좌우 흔들기
* 회전 반복 모션

---

## 뒤돌기 보정

180도 경계 통과 시 발생하는 값 점프를 보정합니다.

예시:

보정 전

179°
-179°

↓

큰 점프로 인식

---

보정 후

179°
181°

↓

연속 회전으로 인식

---

효과:

* 옆면 튐 제거
* 뒤돌기 시 끊김 감소
* 회전 연속성 향상

---

## 임계값

뒤돌기 보정을 시작하는 기준값입니다.

기본값:
270°

값을 낮추면:

* 더 민감하게 감지

값을 높이면:

* 더 큰 변화에서만 감지

일반적으로 기본값 유지 권장.

---

# 범위 설정

## 전체 범위 (도)

Width 모드에서 사용됩니다.

예시:

180°

↓

-90° ~ +90°

---

## 최소 / 최대

Min/Max 모드에서 사용됩니다.

예시:

최소: -30°
최대: +60°

↓

-30° → 0
+60° → 100

으로 매핑됩니다.

---

### 추천 설정

#### 일반 MMD 모션

* Local
* Auto-Range ON
* 90도 반전 OFF
* 뒤돌기 보정 OFF

#### 춤/턴 모션

* Local
* Auto-Range ON
* 90도 반전 ON
* 뒤돌기 보정 ON
* 전체 프레임 정밀 스캔 ON

#### 180도 이상 계속 회전하는 모션

* Local
* Auto-Range ON
* 90도 반전 ON
* 뒤돌기 보정 ON
* 전체 프레임 정밀 스캔 ON
* 임계값 270° 유지
